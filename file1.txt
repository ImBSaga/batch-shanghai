1. git init (Awal Bikin Repository)
2. git status (Cek Status)
3. git add (Tambah File)
4. git commit -m "Menambahkan file1.txt" (Simpan Perubahan)
5. git log (Cek Perubahan)
6. git add . (Tambah Semua File)
7. git clean -f (Hapus File)
8. git restore *namafile* (Mengembalikan File)
9. git restore --staged *namafile* (Mengembalikan File)
10. git log --oneline (Cek Perubahan)
11. Create .gitignore (Mengabaikan File)
12. git branch -M main (Mengubah Nama Branch)
13. git remote add origin <url> (Menambahkan Remote)
14. git remote -v (Cek Remote)
15. git push -u origin main (Mengirim Perubahan) (u = upstream, main = branch)
16. git pull (Menerima Perubahan)
17. git clone <url> (Mengkloning Repository)
